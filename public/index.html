Packed.mobile

Packed design system

Warm, sunset-led React Native components.

A standalone React Native Reusables registry. Add any component straight into your Expo app with the RNR CLI — it lands in components/ui and resolves its dependencies for you. NativeWind, iOS & Android.

Configure

Point the @packed-native namespace at this registry in your app's components.json:

"registries": {
  "@packed-native": "https://mobile.trypacked.dev/r/{name}.json"
}

Install

Add the theme once, then any component:

npx @react-native-reusables/cli add @packed-native/theme
npx @react-native-reusables/cli add @packed-native/button
